Svåra situationer

Att hjälpa en person med demenssjukdom kan vara en stor utmaning. Det är inte alltid lätt att veta var man ska vända sig för att få den hjälp man behöver. Tänk på att du som anhörig enligt lag har rätt till stöd och hjälp. Här beskrivs olika stödfunktioner.

Här kan du information och tipps hur du kan göra i några svåra situationer so kan uppstå till följd av en demenscsjukdom.

Distriktssköterska/läkare

Om din närstående får ett ändrat beteende som är svårt att hantera kan det behövas en medicinsk bedömning. Vänd dig till vårdcentralen eller hemsjukvården.

Kurator

En kurator ger råd, stöd och information och kan även samordna kontakter med andra myndigheter. Du kan även få samtalsstöd. Vänd dig till din vårdcentral för att få kontakt med en kurator.

Arbetsterapeut

Ordinerar och provar ut hjälpmedel samt ger konkreta tips som kan underlätta vardagen för dig och din närstående (elektronisk almanacka, medicinpåminnare, spisvakt eller GPS- larm).

Du kan även få förslag på hur din bostad och omgivning kan anpassas efter era behov (badrum, trösklar).

Anhörigstöd

Anhörigstöd finns i varje kommun. Där kan du få råd och stöd om vad du som anhörig och din närstående har rätt till. Många erbjuder information om demenssjukdomar och ger även samtalsstöd enskilt eller i grupp.

Demenssjuksköterska

Vissa kommuner har demenssjuksköterskor som kan ge information om demenssjukdomar och ge tips på bemötande. De kan också ge stöd vid svåra situationer, t ex om din närstående saknar insikt och inte vill ta emot hjälp. De kan även lotsa dig rätt mellan olika vårdgivare.
